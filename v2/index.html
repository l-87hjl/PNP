What is this?

This project explores the relationship between physical lock mechanisms and Boolean satisfiability (SAT) problems. Each lock consists of numbered dials that can be set to position 1 (FALSE) or 6 (TRUE). Constraints enforce logical relationships between dials, creating an equivalence with 3-SAT problems.

The lock system encodes three types of constraints:

  • Binary Pins (Bin): Restrict dials to positions {1, 6}
  • Negation Links (Not): Enforce diali + dialj = 7 (opposite values)
  • OR Clauses: Enforce diali + dialj + dialk ‚â• 8 (at least one TRUE)

A lock is "unlockable" if and only if the corresponding SAT formula is satisfiable. This establishes a direct connection between physical constraints and computational complexity theory, making the abstract P vs NP question more tangible.

How it Works
